# Marketplace Intelligence Platform v1.0 - Visual Dashboard Pack

This package contains a complete Python-generated dashboard suite designed to showcase business intelligence, Power BI-style reporting, data analysis, customer intelligence, seller risk analysis, network science and decision intelligence using the Olist marketplace dataset.

## Dashboards

1. Platform Hero / Architecture
2. Executive Intelligence Center
3. Customer Intelligence Engine
4. Seller Intelligence & Risk Center
5. Product & Category Intelligence
6. Network Science & Marketplace Fragility
7. Decision Intelligence Engine

## Main Outputs

- `marketplace_intelligence_platform_gallery.html` - interactive visual gallery
- `Marketplace_Intelligence_Platform_v1_Dashboards.pdf` - multi-page PDF dashboard deck
- `visual_dashboards/*.png` - individual 16:9 dashboard images
- `data_exports/*.csv` - recovered tables used to build the dashboard suite

## Business/Portfolio Signal

This is not only a dashboard pack. It demonstrates an analytics platform moving from descriptive BI to prescriptive decision intelligence:

Raw Data -> Data Model -> BI Layer -> Customer Intelligence -> Seller Risk -> Network Science -> Decision Engine.
